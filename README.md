# Saucedemo Automation Framework (POM)

This is a Selenium automation framework for the **SauceDemo** website, built using the **Page Object Model (POM)** design pattern.

## 🚀 Tech Stack
* **Language:** Java 21
* **Automation Tool:** Selenium WebDriver (v4.40.0)
* **Test Framework:** TestNG
* **Reporting:** Extent Reports
* **Build Tool:** Maven

## 🛠️ Features
* **Cross-Browser Testing:** Supports Chrome, Firefox, and Edge.
* **Data-Driven:** Managed via JSON and Properties files.

## 🏃 How to Run
1. Clone the repository.
2. Update `config.properties` with your preferred browser.
3. Run the following command in the terminal:
   ```bash
   mvn test
